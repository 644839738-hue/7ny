# Product Requirements Document — SpriteForge AI

## 项目背景

独立游戏开发者和小型 2D 游戏团队在制作游戏素材时，常面临美术资源不足、外包成本高、风格不统一等问题。SpriteForge AI 旨在通过 AI 生成 + 自动化后处理，降低 2D 游戏素材的制作门槛。

## 目标用户

- 独立游戏开发者（Indie Game Dev）
- 小型 2D 游戏团队（2-5 人）
- Game Jam 参赛者

## 核心需求

1. **文本/参数生成素材** — 用户输入描述文字或选择参数，即可生成对应素材
2. **四类素材支持** — 角色（Character）、道具（Prop）、Tile、UI
3. **像素风输出** — 支持 32×32 / 64×64 像素规格
4. **自动资产化** — 生成后自动处理透明背景、裁剪、尺寸标准化、主体居中
5. **Sprite Sheet** — 多帧素材拼接 + JSON 元数据
6. **Tile 检测** — 3×3 平铺预览 + 边缘一致性评分
7. **导出** — Unity / Godot 可用的 ZIP 素材包

## DEMO 模式

- 默认开启，无需外部 AI API
- 使用内置样例素材模拟完整生成流程
- 确保评审时可复现所有功能

## 非功能性需求

- 代码结构清晰，方便评审
- 每个 PR 只做一件事
- 主分支任意时间点可运行
